# Spotify Clone by Soumyajit

This project is a working clone of Spotify which is made of HTML, CSS and pure JavaScript.

## Features

- Play music
- Pause music
- Skip to next song
- Skip to previous song

## Technologies Used

- HTML
- CSS
- JavaScript
- Tailwind CSS
